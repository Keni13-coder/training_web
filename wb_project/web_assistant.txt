CSS Easing / Cubic-Bezier Generator - для просмотра анимаций


1. резиновость - отношение размера к маштабированию
     --index: calc(1vw +1vh)

2. для изменяевшихся объктов используется 
     will-change: transform;

3. font face подключение шрифтов:
     @font-face {font-family:name;
                src: url(../fonts/raleway-regular.woff2)}
     можно использовать одно имя , используется для написание разных свойст
     подключение происходит по описание конкретного набора всех свойств для загруженного файла, где - regular является насыщенность.


4. align-items - управляет флекс элементами по горизонтальной линии
   justify-content -управляет флекс элементами по по вертикали(в зависимости от главной оси)


5. создание переменных :root{--name}
     так же переменные создаются в селекторах.
     calc() - если есть вычисления.
     через var(--name) добовляем переменные

6. transform: perspective(z) translate3d(x,y,z); двигает объект по осям
          чем блольше perspective(z) тем дальше объект

7. text-transform: - изменяет регистор текста

8. min and max , при min контект будет растягиваться чтобы уместиться
                 при max контент будет растягиваться до придела указанного в max

9.  text-shadow(x,y,z color) - отвечает за добавление тексту тени, X  право и лево в зависимости от + или - и низ вех от Y, Z отвечает за размытие
             z не обязательна,  color цвет тени


10.  flex-direction -  задаёт направление основных осей в контейнере
